# Changelog

All notable changes to this project are documented here. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and versions follow [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Automated tests, linting, dependency review, release notes, and repository maintenance workflows.
- Multilingual project documentation and community contribution files.

### Changed

- Updated `python-multipart` to a secure release.
- Hardened job identifier validation and Docker error responses.

## [1.0.0] - 2026-02-03

### Added

- Local FastAPI web interface for PaddleOCR-VL.
- PDF and image uploads, background processing, live logs, result previews, and ZIP downloads.
- CPU and NVIDIA GPU execution modes.
