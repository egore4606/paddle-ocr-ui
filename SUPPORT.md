# Support

PaddleOCR-VL Local Web UI is a community-maintained personal project and does not provide guaranteed support or response times.

- Use [Discussions](https://github.com/egore4606/paddle-ocr-ui/discussions) for setup questions, ideas, and general help.
- Use [Issues](https://github.com/egore4606/paddle-ocr-ui/issues) for reproducible bugs and accepted feature work.
- Read the [troubleshooting guide](docs/TROUBLESHOOTING.md) before opening a new report.
- Report vulnerabilities only through [private vulnerability reporting](https://github.com/egore4606/paddle-ocr-ui/security/advisories/new).

Do not upload private documents, OCR results, credentials, environment files, or unsanitized logs.
